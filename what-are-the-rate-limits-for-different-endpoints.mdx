---

Please find below a list of endpoints and their corresponding rate limits (calls per minute):

| Operation                                                     | Rate limit (calls per minute) |
| ------------------------------------------------------------- | ----------------------------- |
| Public GetPaywall                                             | Unlimited                     |
| GetPaywall (customer paywall)                                 | 3000                          |
| ProvisionSubscription                                         | 250                           |
| ApplySubscription                                             | 250                           |
| GetEntitlements / GetEntitlement                              | Unlimited                     |
| GetActiveSubscriptions                                        | 3000                          |
| ProvisionCustomer                                             | 250                           |
| ReportEvent                                                   | 1500                          |
| ReportUsage                                                   | 3000                          |
| PreviewSubscription                                           | 3000                          |
| EstimateSubscription                                          | 3000                          |
| EstimateSubscriptionUpdate                                    | 3000                          |
| cancelSubscription                                            | 250                           |
| Getting customer data                                         | 1500                          |
| Updating customers                                            | 250                           |
| Update subscription                                           | 250                           |
| Archive customer                                              | 250                           |
| Migrating subscriptions to the latest plan and add-on version | 250                           |
| Getting available coupons                                     | 500                           |
| Cancel subscription                                           | 500                           |
