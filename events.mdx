## Features

#### feature.created

Occurs whenever a new feature is created.

Example payload 

JSON

`{
  "type": "feature.created",
  "id": "feature-test-id",
  "name": "feature-test-name",
  "unit": "seat",
  "units": "seats",
  "description": "feature test description",
  "featureType": "NUMBER",
  "meterType": "INCREMENTAL",
  "status": "ACTIVE"
}
`

  
#### feature.updated

Example payload
