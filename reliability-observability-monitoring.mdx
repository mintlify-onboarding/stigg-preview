## Reliability

1. Billing & CRM integrations  
   1. Sync retries for network errors  
   2. Notification on sync failures  
   3. Two-way sync, messages are queued and never lost
2. Every change in the system is persisted and sent to your server using a webhook

## Observability

1. Sync errors are visible in the UI
2. Dedicated health dashboard (coming soon)

## Monitoring

1. Error tracking in real-time of the whole stack
2. Infrastructure monitoring and alerts
3. On-call shifts, 24/7 support
